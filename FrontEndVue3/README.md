# AMOD Vue3 Frontend

This folder contains a standalone Vue 3 + Vite + Element Plus frontend for the agricultural machinery operation data management system.

## Run

```bash
npm install
npm run dev
```

The app proxies API calls to `http://127.0.0.1:8000`.

## Main pages

- Login and register
- Home dashboard
- Track list
- Track map with Leaflet
- Data import log page
- User, role, and menu CRUD pages
